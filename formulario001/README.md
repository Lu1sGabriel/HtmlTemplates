    <h2 id="visao-geral">Visão Geral</h2>
    <p>Este projeto consiste em um formulário de cadastro HTML com recursos de validação de campos em tempo real.
        O formulário inclui campos para nome, e-mail, senha, confirmação de senha, descrição, sexo e experiências.
        Ao preencher os campos, a validação é executada para garantir que os dados inseridos sejam válidos antes de
        enviar o formulário.</p>

    <h2 id="tecnologias">Tecnologias</h2>
    <p>O projeto utiliza as seguintes tecnologias:</p>
    <ul>
        <li>HTML5: linguagem de marcação utilizada para estruturar o formulário.</li>
        <li>CSS3: folhas de estilo em cascata utilizadas para definir a aparência visual do formulário.</li>
        <li>JavaScript (ES6): linguagem de programação utilizada para a validação dos campos e interatividade.</li>
    </ul>

    <h2 id="como-usar">Como Usar</h2>
    <ol>
        <li>Faça o download dos arquivos HTML, CSS e JavaScript.</li>
        <li>Abra o arquivo <code>index.html</code> em um navegador da web.</li>
        <li>Preencha os campos do formulário seguindo as instruções de validação em tempo real.</li>
        <li>Quando todos os campos estiverem preenchidos corretamente, clique no botão "Enviar" para enviar o
            formulário.</li>
    </ol>

    <p>Observação: Certifique-se de que os arquivos <code>style.css</code> e <code>script.js</code> estejam na mesma
        pasta do arquivo <code>index.html</code>.</p>
